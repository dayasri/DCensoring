loglike.dep {DCensoring}R Documentation

Loglikelihood for dependent copula

Description

This function takes input as log(parameters),dataframe, colnames of Y = min(T,C) & delta and the distribution of T and C. This returns the loglikelihood for dependent copula.

Usage

loglike.dep(
  lpara = c(1, log(0.5), 2, log(0.5), tau, beta.T, beta.C),
  Cop.fam = "frank",
  df = df,
  y,
  delta,
  T.fam = c("weibull", "lnorm"),
  C.fam = c("weibull", "lnorm"),
  x
)

Arguments

lpara:

starting values of log(parameters) – eg: c(par(T),par(C),par(copula))

df:

Dataframe

y:

columnname of Y=min(T,C) in dataframe. eg: "survivaltimeinmonths"

delta:

columnname of delta in dataframe. eg: "delta"

T.fam:

specify the distribution of x (survival time) – "weibull" or "lnorm"

C.fam:

specify the distribution of y (censoring time) – "weibull" or "lnorm"

Cop.fam:

specify the copula family – "frank","clayton","gumbel","gauss"

x:

covariates or explanatory variables

Value

loglikelihood for dependent copula with specified margins
