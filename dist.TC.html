dist.TC {DCensoring}R Documentation

Distribution of T (Survival)and C (Censoring)

Description

This function takes input survival & censoring data and distribution family of T and C and returns the estimates of the distribution given.

Usage

dist.TC(t, c, T.fam = c("weibull", "lnorm"), C.fam = c("weibull", "lnorm"))

Arguments

t:

survival time

c:

censoring time

T.fam:

specify the distribution of T (survival time) – "weibull" or "lnorm"

C.fam:

specify the distribution of C (censoring time) – "weibull" or "lnorm"

Value

estimates of x and y
